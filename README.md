# Responsive Landing Page

## Overview
A modern and fully responsive landing page developed using HTML, CSS, and JavaScript. The project features a clean user interface, smooth navigation, and an optimized layout for different screen sizes.

## Features
- Responsive Design
- Fixed Navigation Bar
- Smooth Scrolling
- Scroll-based Navbar Effect
- Hero Section
- About Section
- Services Section
- Features Section
- Contact Form
- Footer
- Interactive Hover Effects

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```
PRODIGY_WD_01/
│── index.html
│── style.css
│── script.js
│── README.md
└── assets/
    └── images/
```

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Learning Outcomes
- Building responsive layouts with HTML & CSS
- Creating interactive UI using JavaScript
- Implementing smooth scrolling and navigation effects
- Organizing project structure for deployment
